# Mensagens WhatsApp — Campanha IA
*Copie e cole. Substitua [nome] pelo nome do cliente.*

---

## 1. Enviar proposta (primeiro contato)

```
Olá [nome]! Preparei uma proposta para a sua campanha 2026.
Segue o PDF com todos os detalhes — como funciona, preços e próximos passos.
Qualquer dúvida estou à disposição 🤝
```
*→ Anexar: proposta.pdf*

---

## 2. Após confirmação do teste (R$350 Standard / R$450 Pro)

```
Perfeito, [nome]! Vamos começar com o teste 🎬

Assim que confirmar o pagamento de R$[350 ou 450] via PIX, me mando o comprovante.
PIX: [sua chave pix]

Em seguida te envio um guia rápido de como gravar o vídeo para criar o seu avatar.
```

---

## 3. Após pagamento — enviar guia de gravação

```
Recebi, obrigado! 🙏

Para criar o seu avatar preciso de um vídeo seu de 2 a 3 minutos.
Preparei um guia com tudo que você precisa saber antes de gravar — leva 5 minutos de leitura.
```
*→ Anexar: guia-gravacao.pdf*

---

## 4. Lembrete — aguardando vídeo de origem

```
Oi [nome], tudo bem? Estamos aguardando o seu vídeo para dar início ao avatar.
Quando tiver pronto, é só me mandar por aqui 📱
Qualquer dúvida sobre a gravação pode me chamar!
```

---

## 5. Vídeo de teste pronto — aprovação

```
[nome], o seu vídeo de teste está pronto! 🎉

Acesse pelo link abaixo para ver:
👉 [link do vídeo]

O que achou? Se quiser ajustes no roteiro podemos refazer.
Quando estiver satisfeito, escolhemos o plano mensal e seguimos! 🚀
```

---

## 6. Após aprovação — confirmar plano mensal

```
Ótimo, [nome]! Fico feliz que gostou 🙌

Para continuar, o plano mensal é:
• Standard — R$697/mês (5 vídeos · 7 min)
• Pro — R$897/mês (qualidade superior)

Qual prefere? Assim que confirmar já configuro tudo e você começa a produzir.
```

---

## 7. Acesso ao plano mensal

```
[nome], tudo pronto! Acesse a plataforma pelo link:
👉 [link do projeto]

Para criar um vídeo:
1. Clique em "Criar vídeo"
2. Escreva o roteiro (ou use a IA para escrever por você)
3. Clique em Gerar — pronto em alguns minutos
4. Compartilhe direto no WhatsApp e redes sociais

Qualquer dúvida estou aqui 🤝
```

---

## 8. Acompanhamento mensal

```
Oi [nome]! Como estão indo os vídeos? 📊
Este mês você ainda tem [X] vídeos disponíveis.
Me avisa se precisar de ajuda com roteiro ou qualquer coisa!
```

---

## 9. Renovação mensal (cobrança)

```
Oi [nome]! Passando para lembrar que a renovação do Campanha IA
vence dia [data] — R$[697 ou 897].
PIX: [sua chave pix]
Qualquer coisa é só me chamar 🤝
```

---

## 10. Se o cliente reclamar da qualidade

```
Oi [nome]! Vamos resolver isso.
Me conta — como estava a iluminação e o ambiente quando você gravou
o vídeo original? Às vezes um pequeno ajuste no material de origem
melhora muito o resultado.
Se preferir, gravamos um novo vídeo de origem com as dicas do guia
e refazemos o avatar sem custo adicional.
```
